####爬取头条
***

思路：
1. 请求今日头条搜索页，键入关键字搜索，发现页面的数据是通过Ajax动态加载的，
通过firebug找到请求的url,得到其中的主干
2. 使用firebug拿到ajax请求的参数，通过`urllib.urlencode()`构造ajax请求
3. 解析索引页后拿到详情页的信息，使用`BeautifulSoup`和正则表达式拿到图片的url
4. 通过得到的图片的url请求图片资源，使用`response.content`拿到图片的二进制流
5. 将二进制写入文件进行保存
***

知识点：
1. 使用`urlencode()`方法构造请求
2. 字符串和`json`的转化及其操作
3. 保存网页上图片
4. 使用`{0}/{1}.{2}`构造字符串
